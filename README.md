# PCM Melting OpenFOAM

![OpenFOAM](https://img.shields.io/badge/OpenFOAM-CFD-1F6FEB?style=flat) ![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=flat) ![Status](https://img.shields.io/badge/status-active-brightgreen?style=flat)

Public roadmap repository for phase-change material melting validation using
OpenFOAM's enthalpy-porosity style source modeling.

## Literature basis

See [docs/literature.md](docs/literature.md).

## Planned deliverables

- Gallium melting cavity benchmark.
- Liquid-fraction and melt-front extraction.
- Time-dependent validation plots.
- Optional fin-assisted PCM extension.
